# French learning
This directory contains small codes and Claude artifacts to help me practice French.

* french_connective_drill.jsx
  - a Claude artifact that helps me to practice French connective words from levels A2-C1. The goal is to enhance automation in speaking and writing.
  - There are three parts: Chunk launcher, Sentence fusion, and Expansion move. The **chunk launcher** drills using the connective words as the "sentence-opener". The **sentence fusion** is to take two bare sentences and merge them as quick as possible. The **expansion move** is to give the base sentence one elaboration move such as reason, contrast, or consequence.
  - In addition to this drill, I should also do two other things: **dictgloss** - listening to one dense paragraph once and reconstruct it from memory; as well as **shadowing rich text** - read aloud reading materials at full speed and mimicking the rythms.
  - To use it, i can upload on a Claude conversation and tell Claude to "render this as an artifact". But in practice, I simply run it in the chat that created this artifact (in order to burn some of my own usage if the weekly limit is not reached).
